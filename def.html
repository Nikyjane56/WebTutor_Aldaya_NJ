FLOWCHART AND ALGORITHM

What is flowchart?

A flowchart is a picture of the separate steps of a process in sequential order.
It is a generic tool that can be adapted for a wide variety of purposes, and can be used
to describe various processes, such as a manufacturing process, an administrative or
service process, or a project plan. It's a common process analysis tool and one of the
seven basic quality tools.

What is Algorithm?

An algorithm is a step-by-step analysis of the process,
while a flowchart explains the steps of a program in a graphical way.
Algorithms are used as specifications for performing calculations and data processing.
